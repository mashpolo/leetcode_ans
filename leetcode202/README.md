# 判断是否是快乐数
> 编写一个算法来判断一个数是不是“快乐数”。

> 一个“快乐数”定义为：对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和，然后重复这个过程直到这个数变为 1，也可能是无限循环但始终变不到 1。如果可以变为 1，那么这个数就是快乐数。

```
示例:

输入: 19
输出: true
解释:
12 + 92 = 82
82 + 22 = 68
62 + 82 = 100
12 + 02 + 02 = 1
```

## 解决思路：
1. 循环求平方数，需要保存平方的和到一个set中，防止无限循环


### 最优思路：
1. 考虑O(1)的算法
2. 采用Floyd算法-龟兔赛跑算法，定义两个指针，一个指针跨度大，一个小，如果两者最终相等，再判断该相等数和1的比较
