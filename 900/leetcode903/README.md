# 按奇偶排序数组

> 给定一个非负整数数组 A，返回一个由 A 的所有偶数元素组成的数组，后面跟 A 的所有奇数元素。

> 你可以返回满足此条件的任何数组作为答案。


```
示例：

输入：[3,1,2,4]
输出：[2,4,3,1]
输出 [4,2,3,1]，[2,4,1,3] 和 [4,2,1,3] 也会被接受。


提示：

1 <= A.length <= 5000
0 <= A[i] <= 5000
```

## 解决办法：
1. 参考快排的思路，可以同时使用两个指针前后依次遍历，如果相遇就停止，其中可以交换位置,使用二进制判断奇偶更快
2. 时间复杂度O(n), 空间复杂度O(1)

```python
class Solution(object):
    def sortArrayByParity(self, A):
        i = 0
        j = len(A) - 1
        while i < j:
            if A[i] & 1 > A[j] & 1:
                A[i], A[j] = A[j], A[i]
            if A[i] & 1 == 0: i += 1
            if A[j] & 1: j -= 1

        return A
```
